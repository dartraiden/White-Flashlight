# **White Flashlight**

## Description
This Magisk module enables the white flashlight on Xiaomi Redmi Note 4 Snapdragon. Supports both stock and custom ROMs, as well as Treble-based ROMs.

## Requirements
- Xiaomi Redmi Note 4 Snapdragon
- Magisk v15+
